## US-001: Telefonnummer-Registrierung

**Priority:** MUST
**Linked Requirement:** WA-AUTH-001

### User Story

> As a **new user**
> I want to **register with a mobile phone number and complete verification**
> So that **to securely create an account and access the system**

### Acceptance Criteria

**AC-1:**
- Given: the user is on the registration screen with an unused valid phone number
- When: the user submits the number and enters the correct verification code within the allowed time
- Then: the account is created and the phone number is marked as verified

**AC-2:**
- Given: the user enters an invalid or improperly formatted phone number
- When: the user attempts to submit the registration form
- Then: the system rejects the input and displays a validation message

**AC-3:**
- Given: the user receives a verification code
- When: the user enters an incorrect or expired code
- Then: the system denies verification and allows the user to request a new code
